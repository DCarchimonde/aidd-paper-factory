## Paper 1 current story

Paper 1 argues that molecular property prediction benchmarks should diagnose the train/test split itself, not only report model scores. The split can change target distribution, scaffold composition, chemical similarity, performance gaps, and model rankings.

## Primary target journal

### Journal of Computer-Aided Molecular Design

Recommended role: **Primary target**

Reasons:

- Directly aligned with AIDD, computational drug design, chemoinformatics, structure-property relationships, and machine-learning methods.
- Hybrid journal, so subscription publication can avoid APC.
- Official page reports fast median first decision.
- SCIE indexed according to the journal page.
- The paper's current story fits best if framed as a reproducible retrospective validation / benchmark audit rather than as a traditional predictive-model paper.

Risk:

- The journal states that traditional predictive methods are prioritized when accompanied by experimental evidence.
- Therefore, we must emphasize that this is not a routine predictive model paper. It is a split-diagnostic benchmark audit with retrospective validation across public datasets.

Manuscript framing for this journal:

> This study does not propose a new molecular predictor. Instead, it provides a reproducible diagnostic audit of how molecular benchmark split protocols affect performance interpretation, target shift, chemical similarity, and model ranking.

## Backup target 1

### Chemometrics and Intelligent Laboratory Systems

Recommended role: **Backup if JCAMD rejects quickly**

Reasons:

- Subscription option has no publication fee.
- Official page reports very fast first decision.
- Scope includes statistical, mathematical, computer techniques in chemistry and well-characterized datasets for testing methods/software.
- Our paper can be reframed as a chemometric benchmark-diagnostic method for molecular property prediction.

Risk:

- Less explicitly AIDD/drug-discovery focused than JCAMD.
- Need to emphasize chemometrics, supervised modelling, chemical data, benchmark diagnostics, and reproducible evaluation.

## Not first choice because of APC

### Journal of Cheminformatics

Recommended role: **Strong fit scientifically, but not first because APC applies**

Reasons:

- Excellent scope match for cheminformatics, QSAR, molecular modelling, chemical structure representations, similarity searching, and data mining.
- However, it is fully open access and has an APC.
- Use only if APC waiver, institutional agreement, or future funding is available.
