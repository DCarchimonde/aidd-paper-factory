JMGM Submission Checklist

Manuscript package:

- Main manuscript PDF
- Editable LaTeX source files
- Bibliography files
- Figure PNG files
- Cover letter
- Highlights
- Declaration of competing interests file if requested by the submission system

Before submit:

- Confirm both authors are entered in the submission system.
- Confirm S. Tong is corresponding author.
- Confirm Y. Wang has approved the submitted version.
- Confirm the paper is not under consideration elsewhere.
- Confirm the public repository link is entered in the data availability field.
- Confirm AI-assisted technology use is declared if the system asks.
- Choose standard subscription route unless open access funding is available.

Recommended graphical abstract:

- Use Figure 1 workflow if the system asks for a graphical abstract.
