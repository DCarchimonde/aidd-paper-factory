# Paper 1 Plan

## Tentative Title

A Leakage-Aware Benchmark for Molecular Property Prediction in AI-Driven Drug Discovery

## Core Research Question

Do common random-split evaluations overestimate the real-world generalization ability of molecular property prediction models compared with scaffold splits?

## Initial Datasets

- BBBP
- BACE
- ClinTox
- ESOL

## Initial Models

- Morgan Fingerprint + Random Forest
- Morgan Fingerprint + XGBoost
- Morgan Fingerprint + SVM
- Optional: GCN / GIN / ChemBERTa later

## Key Outputs

- Random split vs scaffold split comparison
- Dataset summary table
- Performance table
- Generalization gap figure
- Reproducibility checklist
